WHAT IS AMP?

The Accelerated Mobile Pages Project (AMP) is an online publishing format, originally created by Google as an alternative to Facebooks Instant Articles. Its a way to optimize web pages for mobile browsing using a special codeAMP-HTML.
The AMP Project is an open-source initiative aiming to make the web better for all. The project enables the creation of websites and ads that are consistently fast, beautiful and high-performing across devices and distribution platforms.

Another essential detail to pay attention for is the use of the Google Analytics tracking script in the free AMP template. As we talked before there shouldn't be any scripts in AMP, so in order to track your Accelerated Mobile Pages instead of the usual tracking script you should use the snippet provided on the Mobirise AMP page replacing <code> UA-XXXXX-Y </code> with your own tracking ID as easy as that!

LEARN MORE
The only applicable extension when it comes to the Mobirise free AMP theme is the Code Editor. It can be used but with caution since if your code does not comply the AMP rules the resulted pages will probably not get validated as AMP and won't be able to take advantage of all the benefits coming along with that. The rules are quite simple and straightforward however you just need to forget for the usage of any external scripts except for the ones included in the AMP library, use some AMP tags instead of the regular HTML ones for example <code><amp-img> </code> instead of the regular <code> <img></code> tag for wrapping your images and forget about referencing any external CSS and JS files. It might sound a bit too much at first but it's actually quite easy and fast to get around, especially having in mind it's all quite well documented over here: https://www.ampproject.org/docs/

What are AMP Benefits?

  • An AMP-valid theme provides a higher page loading speed. It will help to keep visitors that use mobile devices.
  • Google Search loves AMP-valid pages, so it's easier to promote them. For example, only AMP pages can get listed in Google Carousel results.
  • AMP pages are perfect for news presentation, and other information content, without heavy effects and script. It's good for mobile devices. More info: https://www.ampproject.org

AMP themes restrictions

  • AMP Themes are not compatible with Bootstrap Mobirise themes, and you can't use AMP blocks in them. But you still can use them in other AMP themes.
  • You can use the free AMP theme in Mobirise only if you have access to Internet.
  • You can't use Mobirise extensions in AMP themes (except for Code Editor).
  • If you use Code Editor, please note that AMP themes are using special AMP tags. If you use not AMP compatible tags, your page will not be AMP valid. More info...
  • AMP themes cannot use non-AMP scripts, so there are some restrictions on this free AMP template: for example, the parallax effect is missing. If you're going to use some non-AMP scripts, please note that in this case, your page will not be AMP-valid. More info...
  • To use the Google Analytics please use the following amp snippet:  

<amp-analytics type="googleanalytics" id="analytics1">     
<script type="application/json">
{
  "vars": {
    "account": "UA-XXXXX-Y"
  },
  "triggers": {
    "trackPageview": {
      "on": "visible",
      "request": "pageview"
    }
  }
}
</script>
</amp-analytics>

  • You can't use the regular Google Analytics, because it is not AMP valid. More info...
